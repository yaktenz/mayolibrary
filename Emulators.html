PlayStation 1

PlayStation 2

1. Download and install PCSX2.
2. Do a Normal Installation.
3. During the First Time Configuration, you will be asked to Select a BIOS rom. Go to your downloads folder where you downloaded PCSX2, then go inside the PS2-Bios folder. Right click and copy the ‘SCPH-70012.bin’ file
4. Then in the PCSX2 configurator, click “Open in explorer”, right click and paste the bios file into the folder.
5. Click “Refresh list” and you should see the USA Bios loaded up.
6. Click Finish and launch the emulator.
7. To load a game, select “System” > “Boot ISO (full)” > Select the Disk Image File (.iso)

PlayStation 3

1. Download and run RPCS3.
2. You will be asked to update the software, select Yes.
3. Afterwards, click “Open” > Select the game folder.
4. You will be asked for a PS3UPDAT.PUP file. You can download it here.
5. Select the PUP file and the game will be ready to launch.
